# 特殊表情对应模型运动参数说明（论文体）

## 1. 研究目的与适用范围

为保证多模态交互系统在视觉反馈上的一致性与可解释性，本文对面部表情驱动的模型运动参数进行结构化整理，并给出参数物理意义与控制行为说明。该参数体系用于前端四环联动动画模块，在表情控制开启时对环体的上下振幅、旋转速度、相位错位与节奏冲击进行实时调制。通过参数映射方式，系统可将离散表情标签转化为连续运动轨迹，从而在交互层实现“情绪风格可见化”的控制效果。

## 2. 参数定义与控制机理

在参数设计上，系统采用“基础波形 + 脉冲项 + 情绪抖动 + 编排项”的复合模型。基础频率参数 freq 决定动作节奏主频，幅值参数 amp 决定上下位移强度，旋转参数 rotateDeg 决定角速度基线。pulse 与 sharp 用于控制脉冲感和波形尖锐程度，stagger 用于构造四环相位差异。为增强情绪风格表现，系统引入 bobFreq、bobAmp、bobPunch 与 bobBias 作为高频抖动与偏置项；同时引入 choreoAmp 与 choreoSpeed 以调制整体编排幅值和编排速度。该机制使系统能够在保持动作稳定性的前提下表达不同表情的运动特征。

## 4. 参数行为解释（论文体描述）

从节奏特性看，平静状态采用低频低幅的保守参数组合，体现稳定、柔和的视觉反馈；抿嘴微笑在频率与幅值上适度增强，使系统表现出连续且轻快的律动；咧嘴大笑通过提升频率、幅值与脉冲项，形成更明显的动态冲击。惊讶状态进一步提高编排速度与抖动项，并在轨迹中加入额外上抬偏置，以表现突发性与高激活特征。愤怒状态则在较高旋转速度与较尖锐波形参数下，呈现更强烈的方向感与张力。

从执行约束看，系统在旋转控制中采用“完整一圈后再换向”的策略，以避免高频换向导致的视觉抖动与轨迹不连续问题。该约束机制在保证情绪表达强度的同时，提高了动画稳定性与可观测性，对于长期运行场景下的人机交互体验具有积极作用。
