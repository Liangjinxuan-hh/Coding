    语音 / 手势桥接

    页面内置 `CommandHub`：外部模块可调用 window.DripCommandHub.send(command) 或通过 `postMessage` 发送 { type: "dd-command", payload: "上移" } 来触发动作。

    • LD3320 等串口模块：解析语音后调用 send("上移")
    • APDS-9960 手势：识别后映射到上移 / 下移 / 左转 / 右转 / 停止
    • Python / WebSocket 网关：将指令透传到本页面

    也可在控制台执行：
    window.DripCommandHub.send("上移")

    或发送 window.postMessage({ type: "dd-command", payload: "左转" })
